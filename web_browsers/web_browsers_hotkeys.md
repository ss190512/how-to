## Firefox shortcuts
```
Ctrl+Shift+b: Display bookmarks.
Ctrl+Shift+n: Undo closed window.
Ctrl+Shift+t: Undo closed tab.
Ctrl+Shift+w: Close the window(Alt+F4).
Ctrl+Shift+Del: Clear history.
Ctrl+d: Add a bookmark.
Ctrl+j: Display the download window.
Ctrl+n: Open a new window.
Ctrl+t: Open a new tab.
Ctrl+w: Close the tab.
Ctrl+Tab: Move through tabs.
Alt+left/right arrows: Go back/forward a page.
F5: Reload current page.
F11: Switch to the full screen and back.
End: Go to the bottom of the page.
Home: Go to the top of the page.
```
